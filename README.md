Gun Violence Dashboard ReadMe

Our application provides statistics on gun violence worldwide, with a focus on
the United States. This dashboard provides information about gun ownership,
number of gun homicides, and other info for countries around the world. For
the US, we also provide state level data. Finally, users are able able to compare
countries countries on a variety of gun statistics.

To Run Project Locally:
- Run: npm install
- Run: npm start


Libraries we used: 

Bootstrap: 
- In general, we used Bootstrap for styling 
- We used the BootStrap Card to display State level detail.
    - https://getbootstrap.com/docs/4.0/components/card/
- We used the Bootstrap Navigation for the Nav: 
    - https://getbootstrap.com/docs/4.0/components/navbar/

React-ChartJS-2: 
- All pages make use of this library. We used the following documentation to implement:
    - https://www.npmjs.com/package/react-chartjs-2#docs

React-USA-Map: 
- Homepage uses this library for USA Map. Used the following documentation and observed examples:
    - https://www.npmjs.com/package/react-usa-map
    - https://codesandbox.io/s/react-j2ptm4
